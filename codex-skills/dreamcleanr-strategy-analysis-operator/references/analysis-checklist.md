# DreamCleanr Strategy Analysis Checklist

Use this checklist before finalizing strategy updates:

- Ground every major claim in current repo truth or an explicitly cited external source.
- Keep `Community free`, `Pro one-time later`, and `Team later` unless an intentional strategic change is being made.
- Keep macOS shell first and iPhone/iPad companion second.
- Keep cloud, Stripe, auth, analytics, and subscriptions inside future-incubation docs unless runtime implementation is approved.
- Keep public copy aligned with current detector truth.
- Keep [../../DREAMCLEANR_MASTER_STRATEGY.md](../../DREAMCLEANR_MASTER_STRATEGY.md) as the canonical strategy source.
