# DreamCleanr Commercial Incubation Boundaries

Keep these boundaries explicit:

- Current product is local-first, GitHub-first, and preview-first.
- Community remains free.
- Pro remains later and one-time until the canonical strategy changes.
- Team remains later.
- Backend, auth, Stripe, analytics ingest, feature flags, and subscriptions are planning-only in this phase.
- Any future commercial architecture must avoid uploading file contents or turning destructive cleanup into a cloud action.
