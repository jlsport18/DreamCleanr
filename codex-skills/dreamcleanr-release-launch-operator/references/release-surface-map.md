# DreamCleanr Release Surface Map

Current release surfaces:

- `main` -> GitHub Pages
- `v*` tag -> GitHub Release
- `release: published` -> Install Smoke
- scheduled workflows -> Operations Health, Governance Review, Business Review

Required release artifacts:

- wheel
- sdist
- sample cleanup report
- `scripts/install.sh`
- `scripts/update.sh`

Public launch surfaces:

- GitHub release notes
- homepage
- pricing page
- Product Hunt / founder-led launch copy
