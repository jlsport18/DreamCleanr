MCP tool for Claude

Add DreamCleanr as a local tool so Claude can scan and preview cleanup safely.

    }
  }

Workflow

One product, three modes of calm.

Direct control

Run scans, preview cleanups, apply the balanced routine, and render receipts from the terminal.

dreamcleanr scan --mode balanced
dreamcleanr clean --dry-run --mode balanced
dreamcleanr clean --apply --mode balanced

AI assistant access

Give Claude, Codex, or VS Code a safe local DreamCleanr tool without exposing destructive cleanup by default.

python3 -m dreamcleanr.mcp_server

Sleep mode cleanup

Install the daily balanced-safe launch agent and let DreamCleanr prepare the machine before you wake up.

dreamcleanr schedule install --mode balanced
dreamcleanr report --input ~/Library/Logs/DreamCleanr/reports/latest.json

FAQ

Answers for high-intent searchers.

Is DreamCleanr a safe Mac cleaner?

Yes. DreamCleanr is designed around conservative defaults, protected AI state, and receipt-first cleanup so you can see what changed after every run.

Can DreamCleanr help with Docker, Claude, and Codex cleanup on Mac?

That is the main niche. DreamCleanr is built for AI and developer-heavy Macs where stale helpers, cache noise, and workflow clutter build up over time.

Does DreamCleanr auto-delete important files?

No. It protects critical Claude, Codex, and Docker state by default and keeps risky VM storage outside automatic deletion.
